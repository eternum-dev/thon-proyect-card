### Instalación

#### Clonar este repositorio

`git clone https://github.com/eternum-dev/thon-proyect-card`

#### Acceder al archivo Nombre Proyecto

`cd ./thon-proyect-card`




### Scripts del proyecto

#### Instalar Dependencias

`npm install`

#### Iniciar el proyecto

`npm start`

#### Construir para producción

`npm run build`
